# SquadraCorse
Creazione di una full stack application che permette il collegamento tramite un interfaccia react e una macchinina tramite un volante esterno e un raspberry pi sulla macchina.

Comunicazione: 
- Frontend (realizzato con React) -> server Node.js
- server Node.js -> setting.json e map.json
- server NOde.js -> RasberryPi (gestione comandi)
